# Dark Mode Quick Reference

For Braun Design Language verification.

## Core Rules

1. Never pure black (`#000000`) — use `#0a0a0b`
2. Never pure white text — use `rgba(250,250,248,0.87)`
3. No shadows — use surface elevation
4. Maintain 4.5:1 contrast (WCAG AA)

## Surface Elevation

| Level | Hex | Use |
|-------|-----|-----|
| 0 | `#0a0a0b` | Canvas |
| 1 | `#111113` | Cards, sidebars |
| 2 | `#1A1A1C` | Elevated cards |
| 3 | `#252527` | Modals |
| 4 | `#2D2D30` | Tooltips |

## Text Opacity

| Use | Opacity |
|-----|---------|
| Primary | 87% |
| Secondary | 60% |
| Muted | 38% |

## Borders

| Use | Hex |
|-----|-----|
| Subtle | `#1f1f23` |
| Default | `#2D2D30` |
| Emphasis | `#3A3A3E` |

## Accent (Amber)

- Primary: `#fbbf24`
- Hover: `#f59e0b`
- Surface: `rgba(251,191,36,0.15)`
