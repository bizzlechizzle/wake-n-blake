# Component Quick Reference

For Braun Design Language verification.

**FONT: Braun Linear ONLY** - No Inter, no JetBrains, no exceptions.

## Buttons

| Type | Background | Border | Text |
|------|------------|--------|------|
| Primary (light) | `#1C1C1A` | none | `#FFFFFF` |
| Primary (dark) | `#FAFAF8` | none | `#0a0a0b` |
| Secondary | transparent | `#C0BFBC` | `#1C1C1A` |
| Ghost | transparent | none | `#5C5C58` |
| Accent | `#f59e0b` | none | `#1C1C1A` |

**Specs:** 44px min-height, 10px/20px padding, 6px radius, 15px font, 500 weight (Medium)

## Inputs

- Height: 44px
- Padding: 10px 12px
- Border: 1px `#C0BFBC`
- Radius: 6px
- Focus: border `#f59e0b`
- Font weight: 400 (Regular)

## Cards

- Background: `#FFFFFF` (light) / `#1A1A1C` (dark)
- Border: 1px `#E2E1DE` (light) / `#1f1f23` (dark)
- Radius: 8px
- Padding: 16px or 24px

## Typography (Braun Linear)

| Use | Size | Weight |
|-----|------|--------|
| Labels | 11px | 500 Medium (UPPERCASE) |
| Meta/Code | 13px | 300 Light |
| Body | 15px | 400 Regular |
| Subsection | 17px | 500 Medium |
| Section | 20px | 500 Medium |
| Page title | 24px | 700 Bold |
| Hero | 30-36px | 700 Bold |

**Available Weights:** Thin (100), Light (300), Regular (400), Medium (500), Bold (700)

## Spacing

8pt grid: 4, 8, 12, 16, 24, 32, 48, 64px

## Border Radius

- Badges: 4px
- Buttons/inputs: 6px
- Cards: 8px
- Avatars: 50%
